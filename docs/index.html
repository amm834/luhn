Luhn Algorithm

Luhn algorithm that can validate the cc - Credit Card numbers is valid or not.

Usage

 
import Luhn from '@amm834/luhn';
const card_number = 4895048712071025;
const result = Luhn.validate(card_number);
console.log(result); // => true
